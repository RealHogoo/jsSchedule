# 마일스톤 기능 문서

## 목적

마일스톤은 프로젝트의 주요 기준 시점과 산출물 단위를 관리하는 기능이다.
태스크를 일정한 묶음으로 관리하고 프로젝트 진행 단계를 표현하는 데 사용한다.

## 화면 방향

- 프로젝트 내부 전용 페이지로 분리한다.
- 목록과 상세 편집을 중심으로 구성한다.

## 주요 화면

- 마일스톤 목록
- 마일스톤 상세
- 마일스톤별 태스크 보기

## 주요 정보

- 마일스톤명
- 상태
- 목표일
- 진행률
- 연결된 태스크 수

## 운영 원칙

- 마일스톤은 프로젝트 단계 또는 납기 기준으로 사용한다.
- 태스크는 선택적으로 마일스톤에 연결한다.
- 마일스톤은 프로젝트 캘린더와 대시보드에서도 노출된다.

## 고려 사항

- 마일스톤은 과도하게 세분화하지 않는다.
- 운영자가 단계 구분과 주요 일정 확인에 쓸 수 있는 수준으로 유지한다.
